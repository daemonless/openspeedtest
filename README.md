## Deployment

### Podman Compose

```yaml
services:
  openspeedtest:
    image: ghcr.io/daemonless/openspeedtest:latest
    container_name: openspeedtest
    environment:
      - PUID=1000
      - PGID=1000
      - TZ=UTC
    ports:
      - 3005:3005
    restart: unless-stopped
```

### Podman CLI

```bash
podman run -d --name openspeedtest \
  -p 3005:3005 \
  -e PUID=@PUID@ \
  -e PGID=@PGID@ \
  -e TZ=@TZ@ \
```
